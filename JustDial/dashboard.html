How to scrape JustDial — 3 methods
Method A: Bookmarklet (zero install, manual click on JustDial page)
Drag this to your bookmarks bar → Scrape JD
Then on any JustDial results page, click that bookmark → CSV downloads.
Method B: Tampermonkey Script (auto-injects Scrape button)
  1. Install Tampermonkey extension
  2. Open justdial-scraper.user.js → it will prompt to install
  3. Every JustDial results page gets a floating Scrape button → click → CSV downloads
Method C: Automatic scraper (no browser, 100% automated)
node scraper.js --all — scrapes all 432 queries via HTTP (no browser needed).
node scraper.js --serve — starts dashboard server at http://localhost:3456 with live state sync.
node scraper.js --city <name> — scrape a specific city only.
node scraper.js --list-pending — see what's left to scrape.
